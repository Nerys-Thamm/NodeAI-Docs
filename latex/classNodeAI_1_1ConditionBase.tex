\doxysubsection{Detailed Description}
The base class for Condition Nodes. 

Condition Nodes are the base class for all nodes that check conditions. Nodes of this type determine if their condition is true or false. If the condition is true, the node will succeed, otherwise it will fail. Condition nodes are used as part of the control flow of the node tree.
