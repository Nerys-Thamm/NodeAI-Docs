\doxysubsection{Detailed Description}
Has a chance to run the child node. 

The chance is determined by the chance property. When the node is initialised it will use RNG to determine whether or not it will run the child node. If it runs the child node, it will return whatever the child node returns. Otherwise, if it fails the chance test, it will return failure.
