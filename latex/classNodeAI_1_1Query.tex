\doxysubsection{Detailed Description}
Base class for all query nodes. 

A query node is a node which executes some form of logic at runtime which is not directly part of the Agent\textquotesingle{}s behaviour. \mbox{\hyperlink{classNodeAI_1_1Query}{Query}} nodes can have any number of input and output properties, and can be used to either process said inputs in some way and return the result as an output, collect information from the scene, or both. 

The following snippet shows how to create a query node that takes a Color property and returns its intensity as an output\+:
